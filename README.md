# ☕ Sip Studio

> Craft your perfect drink, one sip at a time.

Sip Studio is a cozy café drink customizer web app where users can build their dream drink step by step — choosing their base, milk, syrup, extras, size, and temperature — then place an order and share their creation with friends.

## ✨ Features
- 8-step drink builder with smooth progress tracking
- Floating drink preview that updates as you build
- Order confirmation with receipt-style summary
- Share your drink with a pre-written caption and site link
- Orders saved to a SQL Server database

## 🛠️ Tech Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** SQL Server (SSMS)

## 🚀 Getting Started

### Frontend
```bash
cd sip-studio-frontend
npm install
npm run dev
```

### Backend
```bash
cd sip-studio-backend
npm install
node server.js
```
